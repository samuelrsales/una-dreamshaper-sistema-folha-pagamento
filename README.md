# 💰 Sistema de Folha de Pagamento - Java

Este é um projeto prático desenvolvido em Java para gerenciar e calcular a folha de pagamento de colaboradores de uma empresa hipotética. O foco principal é aplicar os fundamentos da **Orientação a Objetos** e a manipulação de **Listas Dinâmicas**.

## 🎯 Objetivos do Projeto
O sistema permite cadastrar diferentes tipos de colaboradores e calcular o salário final automaticamente, baseando-se nas regras de negócio de cada vínculo empregatício. O programa é interativo e utiliza o console para entrada e saída de dados.

## 🚀 Conceitos Aplicados
Durante o desenvolvimento, foram colocados em prática os seguintes conceitos de Java:

*   **Tipos de Dados:** Uso de primitivos (`int`, `double`) e compostos (`String`, `List`).
*   **Constantes e Variáveis:** Utilização de `final` para valores fixos e variáveis bem escopadas.
*   **Entrada e Saída:** Manipulação de dados com a classe `Scanner` e `System.out`.
*   **Estruturas Condicionais:** Lógica de decisão com `if`, `else if` e `switch case`.
*   **Laços de Repetição:** Controle de fluxo com `for`, `while` e `do-while`.
*   **ArrayList:** Armazenamento dinâmico de objetos na memória.
*   **POO (Orientação a Objetos):**
    *   **Classes e Atributos:** Modelagem das entidades.
    *   **Herança:** Criação de classes filhas a partir de uma classe base.
    *   **Polimorfismo:** Um único método (`calcularSalario`) com comportamentos diferentes.
    *   **Encapsulamento:** Proteção de dados com modificadores de acesso e métodos.

## 🛠️ Funcionalidades
O sistema conta com um menu interativo onde é possível:

1.  🆕 **Cadastrar Funcionário Padrão:** Recebe o salário base fixo.
2.  📈 **Cadastrar Funcionário Comissionado:** Salário base + comissão sobre vendas.
3.  ⚙️ **Cadastrar Funcionário de Produção:** Salário base + bônus por peças produzidas.
4.  📋 **Gerar Folha de Pagamento:** Lista todos os funcionários com seus respectivos salários calculados.
5.  ❌ **Sair:** Encerra a aplicação.

## 💻 Como Rodar o Projeto
1.  Certifique-se de ter o **JDK 11 ou superior** instalado.
2.  Clone este repositório ou copie os arquivos `.java`.
3.  Abra na sua IDE de preferência (IntelliJ, VS Code, Eclipse).
4.  Execute a classe `Main.java`.
